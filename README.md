# actions_build_ErfanGSIs #

基于Github Actions制作的自动跑ErfanGSI的脚本。
## 操作步骤：
1.fork这个仓库
2.进入.github/workflows 更改其中的 build_ErfanGSIs.yml 里的变量 ROM_URL 成 ROM 的 下载直链 ROM_NAME 改成你的 ROM 名称。
3.点击顶部的 star 开始执行
4.在 Upload GSI 里展开，看 Download Link 链接，访问即可下载。
